
# GLUONiCA WS
### Погодная станция

> На базе `wemos mini esp32` и дисплея `WeAct Studio 4.2-Black-White-Red`

> основано на [проекте](https://github.com/G6EJD/ESP32-e-Paper-Weather-Display)

- Показывает погоду из опен-метео 
  -  недельный прогноз температуры
  -  недельный прогноз осадков
 
- микроклимат дома с датчика SCD40 
  - co2 
  - температура
  - влажность 

- можно подключить дополнительно датчики  BME, SHT, GXHT, AHT 

- можно добавить в хом ассистант используя интеграцию mqtt

### Сборка
- в корпусе места для проводов мало, лишнюю длину отрезать
- скрутить провода дисплея косичкой
- припаять к есп - провод сверху пайка снизу, как на фото
- припаять провода к датчику scd40 - сверху пайка снизу провода (как на фото), припаять к есп
- кнопки - общий провод на землю, одна кнопка на пин резет, вторая на пин 25
- полностью спаянное устройство подключить через usb к устройству
  - скачать данный репозиторий, распокавать, открыть проект в вс код с установленным плагином платформио
  - заполнить учетные данные в коде ( в самом вверху константы)
  - выбрать wemos_d1_mini32_GWS_01 - для загрузки
  - выбрать порт загрузки
  - загрузить прошивку
- если после загрузки увидели на дисплее логотип погоднеой станции - можно отключать и собирать все в корпус
- дисплей прикручивается на два самореза
- есп приклеивается к корпусу на двухсторонний скотч.
- вставить датчик scd40 в отверстие, поставить кнопки на место и прижать напечатанной планкой и двумя саморезами
- аккуратно расправить провода и собрать корпус, провода расправить и распределить по корпусу, места мало, все очень плотно
- закрепить верхнюю крышку тремя саморезами
