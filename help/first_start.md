# Настройка

## Интеграция в Home Assistant

Устройство может передавать данные с датчиков в Home Assistant

Для этого должно быть установлено дополнение Mosquitto broker - MQTT (если вы используете zigbee2mqtt то оно уже у вас установлено) 

### Настройка MQTT 
- Перейдите в веб интерфейс погодной станции и активируйте переключатель "тонкие настройки"
- Прокрутите вниз до раздела настроек MQTT и активируйте переключатель MQTT
  - при этом будет произведена попытка подключения, интерфейс может зависнуть на несколько секунд. Немного подождите и перезагрузите страницу.
- в поле сервер введите ip адрес сервера mqtt? например `192.168.1.128`
- в логин - логин созданный при установке добоплнения Mosquitto broker
- пароль - пароль созданный при установке добоплнения Mosquitto broker
- порт - если не меняли при установке добоплнения Mosquitto broker то оставьте `1883`
- имя клиента можно оставить как есть
- подписаться -  ненужно ничего вводить (это поле для получения данных с другого устройства)
- доступные устройства - ненужно ничего вводить (это поле для подписки на данные с другого устройства GLUONiCA)
- отправлять - топик для отправки данных - оставить как есть
- отправлять данные - переключить в положение вкл.
- попыток подключения - сколько неудачных попыток подключения подряд до часового таймаута, 0 - бесконечно (можно оставить как есть)
- Неудачных попыток подключения - сколько было попыток подключения к мктт подряд, закончившихся неудачей.

### Добавление датчиков в HA 
- после успешного подключения, погодная станция отправит данные с датчиков
  - по умолчанию CO2, температура, влажность берется с датчика SCD40, а давление из погоды
- для добавления в панель выберите карточку `сенсор` и ищите по имени устройства
- в данном исполнении, устройство передает только температуру, влажность, C02 и давление. остальные данные всегда 0;
- сенсор Random нужен для определения времени последнего получения данных от устройства. При каждой отправке число меняется.
